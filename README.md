# rewards-bot
Rewards bot collection using [Puppeteer](https://github.com/GoogleChrome/puppeteer).

* 楽天カード
* 楽天銀行
* Yahoo!
* ドットマネー
* Qoo10
* GMOポイント
* ポイントタウン
* げん玉
* ちょびリッチ
* モッピー
* お財布.com
* ワラウ

## Setup

```
$ npm i
```

The installed packages are:

* dotenv
* log4js
* puppeteer
* yargs

## Configuration

Edit `.env`.
